lexico: text analyzer

View the project

Overview

lexico is one of the first projects I made when learning about web development.

It takes any inputted text and analyzes it to return some interesting data, including lexical density (meaningful words divided by total words) and unique occurrences of words. Try it out!

I remember having fun choosing the colors and making the page responsive.

I haven't made any changes to it since I created it in 2017, but I'll periodically check to make sure the CanvasJS API still works.

Some improvements that could be made

  • Make the buttons look more like buttons.
  • Make the text field look more like a text field.
  • Exclude numbers from the "Unique Words" list.
  • Increase the target size of the radio buttons.
  • Filter out non-lexical words by default
