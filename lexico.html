lexico: text analyzer

View the project

Overview

lexico is one of the first projects I made when learning about web development. It was a great way to practice my HTML, CSS, and Javascript skills.

It takes any inputted text and analyzes it to return some data, including lexical density (meaningful words divided by total words) and unique occurrences of words. Try it out!

I remember having fun choosing the colors and making the page (somewhat) responsive.

There's lots of room for improvement :) but I likely won't revisit this project.

Some improvements that could be made

  • Make the buttons look more like buttons.
  • Make the text field look more like a text field.
  • Exclude numbers from the "Unique Words" list.
  • Increase the target size of the radio buttons.
  • Filter out non-lexical words by default.
