lexico - a text analyzer


DEMO

HTML CSS JS

Lexico is a web application that can take inputted text and return some interesting information about the text in graphical and numerical form.


I thought it would be cool to visualize and see the complexity of word choice in various writings such as songs, essays, articles, and posts so I created lexico. I also challenged myself in this project by practicing using only Javascript instead of relying on jQuery.
