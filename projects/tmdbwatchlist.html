TMDb Watchlist


HTML CSS/SCSS JS jQuery API

DEMO

This web app uses the TMDb API and can help users discover trending films and films from different genres which can then be saved to a list and downloaded as a plain text file.


I wanted this project to be a single page application and it was a challenge to manipulate the DOM to load the correct information for each type of movie based on what genre the user selects or what they search. I liked the result, which is being able to go through all the information displayed by the app without having to reload the page.
