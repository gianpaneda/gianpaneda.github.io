Twitch Clip Theater lets users view the most popular clips from the top 100 trending games on Twitch.tv, a live streaming video platform that focuses on video games.


I built this project as a way to practice working with an API and using jQuery, which made the process significantly easier. Getting the clips to display in a (relatively) visually pleasing way was a challenge, as was displaying the game box art and creating a way to scroll through the different games.
